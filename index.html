你可以做什么

这些都来自家长们的真实诉求。立项时导师会一对一帮你把想法收敛到课程内能做完、而且你自己天天会用的范围。

网站

人人都会先做一个。家庭 / 个人品牌网站、烘焙 · 家教 · 民宿等小生意官网、社区活动页面、孩子的作品集网站。

Web 应用

最受欢迎的方向。孩子阅读 / 练琴打卡追踪、家庭日程管理、大学申请选校清单追踪、生意预约与报价小工具。

手机App

装进全家人的手机。家庭共享购物清单、习惯养成打卡、家庭菜谱与每周晚餐计划 — 做完直接安装到手机桌面。

数据分析

做完就能用在工作和生意里。家庭开支可视化、学区学校数据对比、自家生意的销售月报自动化、投资组合回顾。

为什么要学 AI Coding Agent · 当场做出第一个网页

第一个真正的网站，放到互联网上

学会 GitHub — 全世界开发者都在用的代码仓库，并用 GitHub Pages 把你的网站发布上线。改样式、加板块、换图片，迭代两三轮直到满意。

🎯 里程碑：你的网站有了真实网址，发到群里 🎉
03立项

确定你的毕业项目 + 搭起 Full Stack 应用骨架

老师一对一立项面谈，把你的想法变成 AI 听得懂的需求说明书。统一帮你接好数据存储和上线部署，让应用真正能存数据、能在线打开。

🎯 里程碑：你的应用 v0.1 已经在线上
04实战

实战 I：推进核心功能，围着你的问题转

从这节起不再讲新课。开场只讲全班共性问题，其余时间老师和助教轮流一对一帮你解决项目里的具体卡点，确保每人都被深度辅导。

🎯 里程碑：核心功能跑通，数据能保存
05实战

实战 II：进阶与打磨

继续一对一解决问题。学有余力可以按需加进阶能力：接入 AI 功能、数据图表、装进手机。和同学交换试用彼此的作品，列出最后一节课前的待办清单。

🎯 里程碑：作品达到「拿得出手」的完整度

最后一节：正式上线，分享你的作品

给作品做最后的收尾，正式上线。然后是轻松的分享时间 — 给大家看看你做了什么，扫码互相试用试用，欢迎带家人朋友一起来看热闹。

常见问题

我完全不会编程，能学会吗？

能。这门课不教你写代码，教你用中文 / 英文把需求说清楚，让 Claude Code 替你写。只要你日常用过 ChatGPT、Claude、Gemini 或豆包，就具备了全部基础。第 1 节课我们会手把手帮每个人把工具装好、跑起来。

没上过 1.0，可以直接上 2.0 吗？

可以。开课前我们会提供一个 20 分钟的「AI 对话速成」视频帮你补齐基础。

除了学费还有什么花费？

需要自备一台笔记本电脑（Mac 或 Windows 均可，近 5 年内购买即可），以及 Claude 订阅（推荐 Claude Max 约 $100/月，Claude Pro 约 $20/月也可）。上线部署和数据存储均使用免费服务，无其他费用。

每周需要投入多少时间？

线下课 2 小时 + 视频预习约 45 分钟 + 自己动手 1–2 小时，每周合计约 4 小时。投入越多作品越完整，保底 4 小时即可跟上进度。

缺席一次怎么办？

当节要点有录播和讲义，助教提供一对一补讲，微信群随时答疑，不会掉队。建议缺席不超过 1 次（全程共 6 节课）。

课程结束后，我的作品还能继续用吗？

作品完全属于你：账号、网址、代码都在你名下。课程教的就是让你课后能自己继续改进它。
